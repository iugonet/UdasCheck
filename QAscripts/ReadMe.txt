If you want to run all QA scripts, you can run QAscript.sh.

QAscript_all.sh is a bash script that executes all QA scripts listed 
in the QAscript.list and output the results to a file named Result_QA.txt.

How to use QAscript.sh:

1. Set IDL path to TDAS and UDAS.

2. Copy end_tests.pro.new to end_tests.pro.

3. Run QAscript_all.sh.

./QAscript_all.sh
